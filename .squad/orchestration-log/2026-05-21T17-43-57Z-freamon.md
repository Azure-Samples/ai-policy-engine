## Goal

Fix APIM_RESOURCE_ID env var binding bug in ASP.NET Core configuration.

## Issue

`ApimManagementOptions` expects the APIM resource ID at config key `Apim:ResourceId`. The Container App Terraform wiring used `APIM_RESOURCE_ID`, which ASP.NET Core does not translate into the nested config key. The standard convention for nested keys is double underscore: `Apim__ResourceId`.
